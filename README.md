# autoDeer
A python package designed for the running of automated and optimised DEER/PELDOR experiments for pulsed EPR. 

## Dependencies
1) Numpy
2) Xepr API for python 3
2) Scipy
3) DeerLab (https://jeschkelab.github.io/DeerLab/)
